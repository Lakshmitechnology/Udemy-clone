# Udemy Clone

A responsive Udemy homepage clone built using HTML and CSS. This project recreates the basic layout and design of the Udemy website for learning and practice purposes.

## Features

- Responsive navigation bar
- Hero banner section
- Course categories
- Recommended courses section
- Popular topics section
- Popular courses section
- Responsive design for desktop, tablet, and mobile devices

## Technologies Used

- HTML5
- CSS3
- Font Awesome
